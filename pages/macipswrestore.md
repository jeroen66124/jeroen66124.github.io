Restore a soft-bricked Mac with the use of a IPSW file in combination with Apple Configurator, DFU Blaster and a second physical Mac.

1. Download the correspondig IPSW file from [IPSW.me](https://ipsw.me/product/Mac/){:target="_blank"} by choosing your model and version
2. Connect a USB-C to USB-C cable between both devices and use the appropriate DFU port on the soft-bricked Mac, which you can identify with [this](https://support.apple.com/en-us/120694){:target="_blank"} Apple guide
3. Make you life easier by downloading the free version of [DFU Blaster Pro](https://twocanoes.com/products/mac/dfu-blaster/){:target="_blank"}, so you can directly put the bricked Mac in DFU mode without the [frustrating key combination](https://support.apple.com/en-us/108900){:target="_blank"}
4. Open Apple Configurator on the working Mac and drag the IPSW file to the detected DFU Mode Mac to restore the firmware
5. You can choose to format the Macintosh HD or keep all the files, usually you select the latter
